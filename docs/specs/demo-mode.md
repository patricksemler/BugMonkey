# Demo Mode Spec

## Purpose

Demo mode lets users understand BugMonkey without creating production scans or accounts.

## MVP Behavior

- Provide a static or seeded demo report.
- Show realistic issue counts and evidence references.
- Avoid fake claims about live scanning.
- Keep demo data safe and generic.

## UI Requirements

- Demo report page.
- Issue detail examples.
- Export examples when export formats exist.

## Next Step

Add demo report data during the web shell or polish milestone, depending on implementation order.
