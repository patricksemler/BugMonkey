# Design System Spec

## Purpose

BugMonkey should feel like a polished SaaS dashboard, not a flashy AI demo.

## Stack

- Tailwind CSS.
- shadcn/ui.
- Radix UI primitives.
- lucide-react icons.

## Style

- Neutral background.
- One accent color.
- Rounded cards.
- Subtle borders.
- Minimal shadows.
- Consistent spacing.
- 14px to 16px body text.
- Compact tables.
- Clear severity badges.
- Strong empty, loading, and error states.

## Avoid

- Excessive gradients.
- Glassmorphism.
- Giant icon or emoji hero sections.
- Fake testimonials.
- Random huge cards.
- Inconsistent button styles.
- Overly flashy AI aesthetic.

## Required States

Every page should include empty, loading, error, and responsive states.
