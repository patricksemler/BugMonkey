# ADR-0005 - Use shadcn And Tailwind Design System

## Status

Accepted

## Context

BugMonkey needs a clean SaaS dashboard UI with accessible primitives and fast iteration.

## Decision

Use Tailwind CSS, shadcn/ui, Radix UI primitives, and lucide-react icons in `apps/web`.

## Consequences

- UI components remain copy-editable and repo-owned.
- Design consistency must be enforced by local components and docs.
- Avoid flashy AI-demo styling and keep the interface calm and practical.
